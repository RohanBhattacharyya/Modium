# Modium
### A cross-site mod finder for Minecraft.

---

#### Features
- Search for mods by name, author, description, and more.(coming soon)
- Searches modrinth (coming soon) and curseforge.
- View mod details, including downloads, versions, and dependencies.
- View mod files and download them. (stretch goal)
- View mod source code (stretch goal)
